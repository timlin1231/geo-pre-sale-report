# GEO 售前诊断报告页面

## 运行
直接双击打开 `index.html` 即可预览页面。

## 数据替换
报告配置和示例数据都在 `app.js`：
- `reportPrd`：报告结构、标题、模板文案、逻辑规则。
- `reportData`：实际渲染数据（指标、图表数据、机会卡片等）。

按客户项目替换 `reportData` 即可复用同一页面结构。

## 关键能力
- Mustache 风格模板插值：`{{variable}}`
- 缺失数据显示：`N/A`
- 指标差距触发痛点文案（支持阈值与样本量限制）
- 全部核心版块可视化：雷达图、象限气泡图、扩展树、热力图、饼图、仪表盘、优先级矩阵
- 当前内置的是 **Trip.com 携程** 的演示数据（结合截图关键指标 + 合理 mock），适合作为内部能力展示模板
